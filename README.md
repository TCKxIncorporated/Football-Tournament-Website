# Football Tournament Website

A modern **football tournament management web application** built using **React**, **TypeScript**, and **Supabase**.

The project allows users to view and manage football tournaments with a clean UI and a scalable frontend architecture.

---

## Tech Stack

- **Frontend:** React + TypeScript
- **Backend / Database:** Supabase
- **Routing:** React Router
- **Build Tool:** Vite
- **Styling:** CSS

---

## Features

- Browse football tournaments
- Register teams
- Responsive UI
- Client-side routing
- Supabase integration

---

## Getting Started

### Clone the repository

```bash
git clone https://github.com/TCKxIncorporated/Football-Tournament-Website.git
cd Football-Tournament-Website
```

### Install Dependencies
```bash
cd Tournament-App
npm install
```
### Configure environment variables

Create a .env file and add:
```ini
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```

### Run the App
```bash
npm run dev
```
The app will run at http://localhost:5173
